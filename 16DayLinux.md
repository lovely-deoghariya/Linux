# Title: Viewing Files with `more`

---

## (1) Create a file named `manual.txt` with at least 25 lines and open it using `more`.

### Ans:

```bash
nano manual.txt
```

(Add at least 25 lines, then save and exit.)

```bash
more manual.txt
```

---

## (2) Scroll to the next page using the correct keyboard shortcut.

### Ans:

```text
Space
```

---

## (3) Advance one line at a time using the correct key.

### Ans:

```text
Enter
```

---

## (4) Exit the `more` viewer correctly.

### Ans:

```text
q
```

---

## (5) Display your current working directory after finishing the exercise.

### Ans:

```bash
pwd
```
