# Title: Editing Files with `nano`

---

## (1) Create a file named `notes.txt` using `nano` and write five lines about yourself. Save the file and exit Nano.

### Ans:

```bash
nano notes.txt
```

(Type 5 lines, then save and exit.)

```text
Ctrl + O
Enter
Ctrl + X
```

---

## (2) Open `notes.txt` again and add two more lines. Save the changes and exit.

### Ans:

```bash
nano notes.txt
```

(Add 2 more lines.)

```text
Ctrl + O
Enter
Ctrl + X
```

---

## (3) Search for a word inside `notes.txt` using Nano's search shortcut.

### Ans:

```bash
nano notes.txt
```

Inside Nano:

```text
Ctrl + W
```

Type the word and press:

```text
Enter
```
